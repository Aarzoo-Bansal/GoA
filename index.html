2.   For each vertex v ∈ G.V:
3.      MAKE-SET(v)
4.   sort the edges of G.E into non-
     decreasing order by weight
5.   For each edge (u, v) ∈ G.E ordered by
     increasing order by weight(u, v): 
6.       if FIND-SET(u) ≠ FIND-SET(v):
7.           A = A ∪ {(u, v)}
8.           UNION(u, v)
9.   return A
